# 链表中等：环形链表入口

## 题目

给定一个单链表，如果链表有环，返回环的入口节点；如果没有环，返回 `NULL/nullptr`。

示例：

```text
1 -> 2 -> 3 -> 4
     ^         |
     |_________|
入口是节点 2
```

## 思路

使用快慢指针：

1. 慢指针每次走一步，快指针每次走两步。
2. 如果有环，快慢指针一定会在环内相遇。
3. 相遇后，让一个指针回到头节点，另一个留在相遇点。
4. 两个指针都每次走一步，再次相遇的位置就是环入口。

## 复杂度

时间复杂度：`O(n)`  
空间复杂度：`O(1)`

## 面试口述

先用快慢指针判断是否有环。相遇后，一个指针从头开始，一个从相遇点开始，它们同步走，第二次相遇就是环入口。
