# 数组简单：二分查找

## 题目

给定一个升序数组和目标值 `target`，如果找到目标值，返回下标；否则返回 `-1`。

## 思路

二分查找每次查看中间元素：

1. `nums[mid] == target`，直接返回。
2. `nums[mid] < target`，说明目标在右半边。
3. `nums[mid] > target`，说明目标在左半边。

## 复杂度

时间复杂度：`O(log n)`  
空间复杂度：`O(1)`

## 面试口述

二分查找适合有序数组。注意 `mid` 不要写成 `(left + right) / 2`，大数组时可能溢出，建议写成 `left + (right - left) / 2`。
