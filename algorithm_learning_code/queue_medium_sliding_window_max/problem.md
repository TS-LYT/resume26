# 队列中等：滑动窗口最大值

## 题目

给定数组和窗口大小 `k`，窗口每次向右滑动一格，返回每个窗口的最大值。

示例：

```text
nums = [1,3,-1,-3,5,3,6,7], k = 3
输出 = [3,3,5,5,6,7]
```

## 思路

使用单调队列，队列里保存数组下标，并保持对应值从大到小：

1. 新元素进入前，把队尾比它小的元素弹出。
2. 新元素下标入队。
3. 如果队头下标已经滑出窗口，就弹出队头。
4. 队头始终是当前窗口最大值下标。

## 复杂度

时间复杂度：`O(n)`  
空间复杂度：`O(k)`

## 面试口述

单调队列能维护窗口最大值。每个元素最多入队一次、出队一次，所以总复杂度是线性的。
