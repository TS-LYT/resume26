# 队列中等：网格 BFS 最短路

## 题目

给定一个 0/1 网格，`0` 表示可走，`1` 表示障碍。从左上角走到右下角，每次只能上下左右移动一格，求最短步数。不可达返回 `-1`。

## 思路

BFS 使用队列：

1. 起点入队，距离为 0。
2. 每次取队头位置。
3. 枚举上下左右 4 个方向。
4. 如果新位置合法、没访问过、不是障碍，就入队。
5. 第一次到达终点时，就是最短距离。

## 复杂度

时间复杂度：`O(rows * cols)`  
空间复杂度：`O(rows * cols)`

## 面试口述

无权图最短路用 BFS。队列保证按距离从近到远扩展，所以第一次到终点就是最短步数。
