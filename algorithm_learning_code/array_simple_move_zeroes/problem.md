# 数组简单：移动零

## 题目

给定一个整数数组，把所有 `0` 移动到数组末尾，同时保持非零元素的相对顺序。

示例：

```text
输入:  [0, 1, 0, 3, 12]
输出:  [1, 3, 12, 0, 0]
```

## 思路

使用双指针：

1. `write` 指向下一个非零元素应该写入的位置。
2. 从左到右扫描数组。
3. 遇到非零元素就写到 `write`，然后 `write++`。
4. 扫描结束后，从 `write` 到末尾全部补 `0`。

## 复杂度

时间复杂度：`O(n)`  
空间复杂度：`O(1)`

## 面试口述

这题可以用双指针原地完成。一个指针扫描数组，一个指针记录非零元素写入位置。先把所有非零元素稳定地搬到前面，再把后面补零。
