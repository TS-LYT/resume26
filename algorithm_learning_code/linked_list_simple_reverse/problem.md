# 链表简单：反转链表

## 题目

给定单链表头节点，把链表反转并返回新的头节点。

示例：

```text
1 -> 2 -> 3 -> NULL
反转后:
3 -> 2 -> 1 -> NULL
```

## 思路

使用三个指针：

1. `prev` 表示已经反转好的前半部分头节点。
2. `cur` 表示当前正在处理的节点。
3. `next` 临时保存 `cur->next`，防止断链后找不到后续节点。

## 复杂度

时间复杂度：`O(n)`  
空间复杂度：`O(1)`

## 面试口述

反转链表就是逐个改变节点的 `next` 指向。每次先保存下一个节点，再把当前节点指向前一个节点，最后整体向后移动。
