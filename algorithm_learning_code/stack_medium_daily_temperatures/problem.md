# 栈中等：每日温度

## 题目

给定每天的温度，返回每一天需要等待多少天才会有更高温度。如果之后没有更高温度，填 `0`。

示例：

```text
temperatures = [73,74,75,71,69,72,76,73]
输出 = [1,1,4,2,1,1,0,0]
```

## 思路

使用单调栈，栈里保存“还没找到更高温度的日期下标”：

1. 从左到右扫描温度。
2. 当前温度高于栈顶日期的温度时，说明栈顶日期等到了更高温度。
3. 弹出栈顶，计算等待天数。
4. 当前日期入栈。

## 复杂度

时间复杂度：`O(n)`，每个下标最多入栈一次、出栈一次。  
空间复杂度：`O(n)`。

## 面试口述

这题用单调栈。栈里维护还没找到答案的下标，当遇到更高温度时，就不断弹出比当前温度低的下标并填写答案。
