# 第 2 阶段：array 固定数组

## 概念

`array` 是 C++ 标准库封装的固定大小数组，大小在编译期确定，不能动态扩容。

## 编译运行

```bash
cmake -S . -B build
cmake --build build
./build/demo
```

## 本示例覆盖

`size`、`at`、`[]`、`front`、`back`、遍历、修改元素、计算平均值。

## 常见错误

1. 以为 `array` 可以 `push_back`。
2. 下标越界访问。
3. 函数参数写成值传递，导致不必要拷贝。

## 嵌入式注意

数据数量固定时优先考虑 `array`，它不需要运行时扩容，行为更可控。

## 面试回答

array 是固定大小数组，比 C 数组多了标准库接口；vector 是动态数组，可以增长，但可能涉及动态内存分配。
