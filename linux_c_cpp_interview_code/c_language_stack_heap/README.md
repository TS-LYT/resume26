# C 语言栈、堆、全局区、static

## 常见内存区域

```text
栈 stack:
  局部变量、函数参数、返回地址。
  自动申请和释放，空间有限。

堆 heap:
  malloc/calloc/realloc/new 申请的内存。
  需要 free/delete 释放，否则可能内存泄漏。

全局/静态区:
  全局变量、static 变量。
  程序启动时分配，程序结束时释放。

代码区:
  存放机器指令，通常只读。

只读常量区:
  字符串字面量等常量数据。
```

## 面试口述

局部变量通常在栈上，函数返回后自动失效；`malloc` 申请的内存在堆上，需要程序员手动释放。`static` 局部变量生命周期贯穿整个程序，但作用域仍然在函数内部。

## 常见错误

1. 返回局部变量地址。
2. malloc 后忘记 free。
3. free 后继续使用指针。
4. 重复 free。
5. 栈上放超大数组导致栈溢出。
