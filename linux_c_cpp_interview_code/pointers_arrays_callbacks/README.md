# 指针、二级指针、数组指针、指针数组、回调函数

## 指针

指针保存地址。`int* p` 表示 `p` 指向一个 `int`。

## 二级指针

`int** pp` 表示 `pp` 指向一个 `int*`。常见用途：

```text
1. 在函数里修改外部指针。
2. 表示二维动态数组。
3. 处理 argv 这类字符串数组。
```

## 数组指针

`int (*p)[3]` 表示 `p` 是一个指针，指向“包含 3 个 int 的数组”。

## 指针数组

`int* arr[3]` 表示 `arr` 是一个数组，数组里每个元素都是 `int*` 指针。

## 回调函数

回调函数就是把函数地址作为参数传给另一个函数。常见于：

```text
1. qsort 比较函数
2. 事件驱动框架
3. Mongoose HTTP 回调
4. pthread_create 线程入口
5. 驱动或业务模块注册处理函数
```

## 面试口述

数组指针重点看括号，`int (*p)[3]` 先和 `*` 结合，说明 p 是指针；指针数组 `int* p[3]` 先和 `[]` 结合，说明 p 是数组。
